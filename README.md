# VSCorzious Visual Studio Code (VSC) Theme

## Installation Guide
### Windows
Clone or download this repo and place the directory at

*`C:\Program Files\Microsoft VS Code\resources\app\extensions.`*

### OSX
Clone or download this repo and place the directory at

*`~/.vscode/extensions`.*
